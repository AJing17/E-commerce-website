# E-commerce-website

A E-commerce-website In iSpan Java
須注意:SMTP 須自己申請，檔案內為範例。
後臺登入因為限制問題:使用前可先取消攔截器，直接進入觀看。
前台一樣會有登入問題:同上。

使用技術:

使用語言:Java

前後台註冊 :  BCrypt 密碼加密

前後台登入 :  Jwt token認證

前後端傳送資料 : Ajax、fetch-Api、springBoot

資料庫 : SQL Server

前端 : html、css、Jquery、Bootstrap
